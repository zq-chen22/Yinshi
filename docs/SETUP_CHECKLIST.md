# 上线验收清单

这份清单用于每一台新主机。不要因为某台机器已经成功，就复制其 Secret、数据库或配对状态。

## A. 规划（0%—10%）

- [ ] 明确机器名、用途、owner 和允许的工作目录
- [ ] 选定唯一机器人名，例如 `Codex-鼎好主机4090`
- [ ] 选定唯一群后缀，例如 `-鼎好主机4090`
- [ ] 确认这台机器不会与另一机器共用 App ID/Secret
- [ ] 决定 Default 还是 Full Access/YOLO，并记录风险接受理由

## B. 飞书应用（10%—35%）

- [ ] 在“因时而得谷”创建企业自建应用
- [ ] 添加 Bot 能力
- [ ] 记录 App ID，未将 App Secret 发到聊天或 Git
- [ ] 添加消息接收、发送、更新、资源和群管理所需权限
- [ ] 已开通 `im:chat:update`，允许自动维护已有群描述
- [ ] 事件订阅选择长连接并添加 `im.message.receive_v1`
- [ ] 回调订阅选择长连接并添加 `card.action.trigger`
- [ ] 可用范围包含 owner
- [ ] 创建版本（首次可为飞书应用 `1.0.0`）并成功发布
- [ ] 在飞书中能够搜索并打开机器人私聊

## C. 主机前置（35%—50%）

- [ ] Python 3.12+、git、venv、bubblewrap、AppArmor 工具已安装
- [ ] `bwrap-userns-restrict` profile 已加载
- [ ] 在运行桥的同一普通用户下，Codex CLI 已安装和登录
- [ ] `codex --version` 正常
- [ ] 手工 Codex 测试任务成功
- [ ] 主机可主动访问飞书和 Codex 所需网络

## D. 桥安装（50%—65%）

- [ ] 克隆 `zq-chen22/Yinshi`
- [ ] 在 `codex-feishu-bridge` 运行安装脚本
- [ ] `config.toml` 填入本机 App ID、群后缀和工作区根
- [ ] `secrets.env` 只在本机填入 App Secret，权限为 `0600`
- [ ] 配置和状态目录权限不向其他用户开放
- [ ] `doctor` 除尚未配对项外无阻塞错误
- [ ] `init` 成功登记本机最近持久顶层对话
- [ ] `recent` 输出符合预期且不包含临时/子代理 thread

## E. 配对和绑定（65%—78%）

- [ ] 运行 `pair-code` 生成短时码
- [ ] 前台运行 `run`，长连接成功
- [ ] 只在对应机器人 P2P 私聊发送配对码
- [ ] 机器人确认 owner 配对成功
- [ ] `bootstrap` 或自动同步创建/认领对话群
- [ ] 每个群绑定正确 thread，群名带唯一机器后缀
- [ ] 每个群描述显示正确工作目录、对话开始时间和 `codex-thread:` 标签
- [ ] 重复 bootstrap 不会创建重复群

## F. 功能验收（78%—92%）

- [ ] 私聊 `帮助`、`状态`、`最近`、`额度` 正常
- [ ] 私聊随机问题收到确认、进度和最终结果
- [ ] 群内普通任务进入正确 thread
- [ ] `!status`、`!steer`、`!stop` 行为正常
- [ ] `/model` 动态显示当前模型目录
- [ ] `/fast` 只在模型支持时可用
- [ ] `/permissions` 三个预设可切换
- [ ] `/compat` 能报告或修复当前 CLI 兼容状态
- [ ] 图片/视频/文件先暂存，下一条文字后一起提交
- [ ] 本轮专用 outbox 的小文件无需额外审批即可回传
- [ ] 任意已有路径、超限或疑似秘密文件不会自动外发
- [ ] 飞书卡片按钮和对应文本兜底均有效

## G. 恢复与运维（92%—100%）

- [ ] 短暂断网后长连接和发件箱自动恢复
- [ ] 服务重启后绑定、附件暂存和队列不丢失
- [ ] 模拟不确定边界时不会自动重放，`待确认` 可见
- [ ] systemd 用户服务已 enable 且 active
- [ ] 如需无人登录运行，linger 已明确启用
- [ ] 重启整机后服务自动启动，手机端再次验收
- [ ] 已建立加密备份和恢复流程
- [ ] 已记录 App Secret 轮换、主机退役和事故响应负责人
- [ ] 已保存当前源码提交、桥版本、Codex CLI 版本和飞书应用版本

全部完成后，这台主机才算完成端到端部署。功能可用但未完成断网、重启、重复执行和秘密保护测试，只能视为试运行。
